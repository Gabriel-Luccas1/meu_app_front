# Front-end PLANEJAMENTO SEMANAL de canteiro de obras

**Desenvolvimento Full Stack Básico** 

O objetivo central deste projeto é viabilizar um planejamento semanal eficiente para obras, permitindo o registro de informações essenciais como nome do encarregado, equipe designada, função específica da equipe, tipo de serviço a ser realizado e a localização correspondente. Essa abordagem proporciona uma gestão precisa e estruturada das atividades ao longo do tempo.

---
## Como executar

Basta fazer o download do projeto e abrir o arquivo index.html no seu browser.
